1. Install the chart

   helm install <release-name> ./h2oai-flood-prediction --namespace <ns> --create-namespace

2. Common overrides

   - Change image tag:
     --set image.tag=v0.2.1

   - Use a different host:
     --set ingress.hosts[0].host=my.domain

   - Disable ingress:
     --set ingress.enabled=false

   - Provide secrets (plaintext):
     --set env.secrets.APP_NVIDIA_API_KEY=abc123 --set env.secrets.H2OGPTE_API_KEY=xyz

3. Registry auth

   If you already have a secret, set imagePullSecrets:
     --set imagePullSecrets[0].name=my-secret

   Or let the chart create one by providing dockerconfigjson:
     --set registrySecret.enabled=true \
     --set-file registrySecret.dockerconfigjson=/path/to/.docker/config.json

4. Uninstall

   helm uninstall <release-name> --namespace <ns>
